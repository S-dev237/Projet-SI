# Django Messenger Project

This is a Django-based messenger application scaffolded for development in Visual Studio Code.

## Getting Started

1. Create a virtual environment:
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Start the Django project:
   ```bash
   python manage.py runserver
   ```

## Project Structure
- `messenger/` - Main Django project folder
- `chat/` - Initial app for chat functionality

## Development
- Recommended to use VS Code with Python and Django extensions.
